## v1 — Le Copilote
Un chat de bureau qui :
- 💬 repond (IA gratuite **Groq / Llama**)
- 🧠 se souvient des conversations
- 🚫 ne se repete jamais (anti-radotage)
- 🎮 aide sur le code de mes projets (Godot, Roblox...)

## Lancer
```
pip install -r requirements.txt
streamlit run app.py
```
